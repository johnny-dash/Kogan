# Installation & Usage

1.  Make sure Node.Js are installed
2.  `git clone https://github.com/johnny-dash/Kogan.git`
3.  `cd Kogan`
4.  `npm i`

## Run

`node index.js`

## Test

run the following command line to run the unit test

`npm run test`

run the following command line to check the unit test coverage

`npm run test:cover`
